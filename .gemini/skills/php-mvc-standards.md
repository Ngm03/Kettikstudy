# СТАНДАРТЫ PHP И АРХИТЕКТУРЫ

Проект "KETTIK STUDY" использует кастомную MVC-архитектуру. 
Ты обязан соблюдать следующие инженерные правила:

1. **Никакого inline-роутинга:** Вся логика маршрутизации должна быть строго инкапсулирована внутри выделенных контроллеров (например, LeadController, PageController). Файл index.php служит только точкой входа.
2. **Безопасность БД:** Все взаимодействия с MySQL осуществляются ИСКЛЮЧИТЕЛЬНО через PDO с использованием подготовленных выражений (prepared statements). Никаких прямых конкатенаций строк в SQL-запросах.
3. **Разделение ответственности:** Контроллеры обрабатывают HTTP-запросы и вызывают Модели. В Контроллерах не должно быть HTML-кода. В Представлениях (Views) не должно быть запросов к БД.
